# Todo List App

A command-line Todo List application built with Go. Focuses on clean project structure, data management, and core backend logic.

---

## Features

- Add, view, update, and delete todo items
- Persistent state management within a session
- Clean project structure following Go conventions

---

## Project Structure

```
Todo-List/
└── TodoList App/
    └── main.go     # Application entry point and logic
```

---

## Getting Started

### Prerequisites

- [Go 1.21+](https://golang.org/dl/)

### Run Locally

```bash
# Clone the repository
git clone https://github.com/Oluwadeuce/Todo-List.git
cd Todo-List

# Run the app
go run "TodoList App/main.go"
```
